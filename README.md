# AltTab - Chrome Tab Switcher

像 Windows Alt+Tab 一样快速切换 Chrome 标签页。

## 快捷键

| 操作 | 快捷键 |
|------|--------|
| 打开选择器 | `Alt+Q` |
| 下一个标签 | `Tab` / `→` / `↓` |
| 上一个标签 | `Shift+Tab` / `←` / `↑` |
| 确认跳转 | `Enter` 或松开 `Alt`/`Q` |
| 取消 | `Esc` 或点击遮罩 |
| 跳到首个/末尾 | `Home` / `End` |

按住 `Alt` 不放，反复按 `Q` 可连续切换；松开即跳转到当前选中的标签。

快捷键可在 `chrome://extensions/shortcuts` 自定义。

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目目录
